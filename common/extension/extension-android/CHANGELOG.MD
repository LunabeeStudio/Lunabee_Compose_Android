# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

# Deprecated, please update the main Changelog

## [1.10.3] - 2020-02-03
## Fixed
- Condition to lift app bar in nested scroll view was inverted

## [1.10.2] - 2020-01-07
## Added
- Extensions `Calendar.getMonth` and `Calendar.setMonth`

## [1.10.1] - 2019-12-30
## Fixed
- Gmail not filling email body when calling Context.startEmailIntent
- Remove warnings by changing the return type of Context.fontFromAttribute, 
TypedArray.getDrawableOrThrow and TypedArray.getFontOrThrow to nullable

## [1.10.0] - 2019-11-27
## Added
- Extension fun `View.hideSoftKeyBoard`

## [1.9.2] - 2019-11-21
## Added
- Years setter and getter: `Calendar.getYear` & `Calendar.setYear` 

## [1.9.1] - 2019-10-24
## Added
- Add recommended `getDrawable` on `TypedArray`

## [1.9.0] - 2019-10-21
### Added
- This changelog
- String extension functions `String.remove(regex: Regex): String` and `String.remove(string: String): String`.
